How About Plotly?

Feb 2, 2019 3 min read

Let’s try some plotly

library(dplyr)
library(ggplot2)
library(plotly)

p1 <- mtcars %>%
 mutate(Cylinders = as.factor(cyl)) %>%
 ggplot(., aes(x=hp, y=mpg, col=Cylinders) ) + 
  geom_point(size=4, alpha=0.7) + 
  theme_bw()

ggplotly(p1)
